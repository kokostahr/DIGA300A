Week Cix: Reflections while incorporating the first few CSS elements into the website

by Star Nkosi

Published: 22 March 2025
Last updated: 14 April 2025

In this blog post, Star shares a reflection on the first few iterations of incorporating CSS elements into her website.

Please note that a deeper, juicier reflection regarding this process can be found both within the style guides section (for a reflection that is more centred on visual elements) and the design reflections section for a reflection that is more focused on the thought process and thoughts regarding the website with the added CSS elements.

I wanted to see how the colours would look like on my actual website so after learning a few tricks, (very few, mind you) I applied the colours to the Index homepage, using my wireframe as a guideline. So far, I am very happy with the colour scheme I have chosen, as all the colours work extremely well together.

Then I started playing around with the fonts to see which fonts felt simplistic yet personally resonated with me.

As you can see from the image, the fonts were…all over the place. This was also the moment where I realised that there were changes that I would have to make to my html structure so that it can match the layout of my wireframe.

In my wireframes, the name of my website is situated on the left-hand side of the navigation bar, so that is a change I will have to make to all the navigation bars across my webpage. Secondly, I will have to change the ‘blog homepage’ link to ‘blogs’ because it is too long to fit within the boxes that I will make once I implement the flexbox style into my navigation bar.

Despite this, I still tried to play around with the fonts, until I finally settled on fonts that worked well together, and with the theme of my website.

I focused on further developing my style guide, updating it with decisions on fonts, colours, and the overall visual direction of my website. I wanted to ensure that every element contributes to the celestial, dreamy atmosphere I’m aiming for. For a more detailed look at my thought process and final choices, check out the Style Guide in my Design section.
