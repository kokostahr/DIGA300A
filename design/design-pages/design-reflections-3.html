Coding the Back To Top Button

I started off by using a tutorial on youtube to frame my foundation This is the tutorial I started off with. Even though I followed this tutorial, a part of me couldn‚Äôt help but feel that it was an outdated method, which in the end turned out to be UNTRUE. At first it did not work out and I couldn‚Äôt really understand why. After getting some feedback, I decided to try the method by making my back to top button, an actual button in the HTML, and by calling the function within the button, in my JS file, in hopes that it would actually workout that time.

For reference, this is how I initially created my back to top button in the HTML code.

And this is how I did the JavaScript code, following the tutorial I watched. In the end, I put the code into a function and changed the anchor tag in the HTML to a button. Used this site for assistance. In the end, I realised that none of my functions were working out because I did not store them in a DOMcontentloaded element. This means that my JS was running at the same time my HTML and CSS were running (when the page is refreshed), so the JS wouldn‚Äôt find the necessary elements in the HTML because they had not loaded yet. The DOMcontentloaded element-function, stopped this from happening by enusuring that the JS functions I did, waited until the HTML pages were done loading.

The html coding for the back to top button did not really change in the end; I just chaned the anchor tag to a button tag.

And added a few more tricks to the JavaScript code so that the button would be inactive when the user is at the top of the page, then appear as they scroll down. If you look at the code, its much longer now. The image below, shows just how much the back to top button changed. Glad I could code it in JS.
